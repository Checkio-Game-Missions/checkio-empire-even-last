Δίνεται ένας πίνακας ακεραίων. Σκοπός είναι να υπολογίσεις το άθροισμα των στοιχείων που βρίσκονται σε άρτιες θέσεις(0, 2, 4...) και να το πολλαπλασιάσεις με το στοιχείο της τελευταίας θέσης του πίνακα. Μην ξεχάσεις ότι το πρώτο στοιχείο βρίσκεται στη θέση 0.

Για τον κενό πίνακα το αποτέλεσμα θα είναι πάντα 0.

Είσοδος: Μια λίστα ακεραίων.

Έξοδος: Ένας ακέραιος αριθμός.

Παράδειγμα: 

checkio([0, 1, 2, 3, 4, 5]) == 30
checkio([1, 3, 5]) == 30
checkio([6]) == 36
checkio([]) == 0

Πώς χρησιμοποιείται: Οι δείκτες είναι σημαντικά θέματα στον προγραμματισμό σε Python, αλλά και στις υπόλοιπες γλώσσες. Θα σου φανούν ακομά πιο χρήσιμα όσο προχωράς.

Προϋποθέσεις: 0 ≤ len(array) ≤ 20
all(isinstance(x, int) for x in array)
all(-100 < x < 100 for x in array)

Το CheckiO δεν είναι απλά η επίλυση αποστολών, αλλά η ανταλλαγή εμπειριών. Λύσε το πρόβλημα, δες τις λύσεις των άλλων χρηστών, δημοσίευσε τις δικές σου απαντήσεις και κάνε σχόλια στις λύσεις των υπολοίπων!
