Se te da un arreglo de enteros. Deberías encontrar la suma de los elementos con índices pares (0, 2do, 4to... ) luego multiplicar ese resultado con el elmento final del arreglo. No olvides que el primer elemento tiene el índice 0.

Para un arreglo vacio, el resultado siempre será 0 (cero).

Entrada: Una lista de enteros.

Salida: El número como un entero.

Ejemplo: 

checkio([0, 1, 2, 3, 4, 5]) == 30
checkio([1, 3, 5]) == 30
checkio([6]) == 36
checkio([]) == 0

Cómo se usa: Índices y fragmentos son elementos importantes de la programación en python y otros lenguajes. Esto se volverá útil en el camino más adelante!

Precondición: 0 ≤ len(array) ≤ 20
all(isinstance(x, int) for x in array)
all(-100 < x < 100 for x in array)
